🟤 BronzePawn

BronzePawn is an addon for World of Warcraft 3.3.5a Special for Acension Bronzebeard Server that helps you quickly evaluate gear based on your own build.

It is based on Pawn, but has been adapted, restructured, and visually redesigned for Bronzebeard.

✨ Features Item evaluation directly in tooltips Upgrade/Downgrade percentage display Fully customizable stat weights Multiple profiles support (e.g. PvE, PvP, Tank, Heal) Custom BronzePawn UI design (bronze/gold theme) 🧠 How it works

BronzePawn calculates an item’s value based on stat weights defined by you.

Example:

Strength = 1.8 Crit = 1.5 Hit = 2.0

The addon sums these values and shows whether an item is better or worse than your currently equipped gear.

⚙️ Installation Download the repository or ZIP file

Extract it into:

World of Warcraft/Interface/AddOns/ Make sure the folder is named BronzePawn Restart the game 🎮 Usage

In-game command:

/bp Opens the configuration window Set your stat weights Compare items directly via tooltips 🔧 Background

BronzePawn was created to provide a cleaner and better adapted version of Pawn for Bronzebeard.

Goals:

Keep the familiar Pawn functionality Improve structure and usability Introduce a distinct visual identity 📌 Notes Accuracy depends on your stat weights For specific builds, manual tuning is recommended 👤 Author
